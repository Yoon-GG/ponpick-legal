PonPick プライバシーポリシー

施行日: 2025年11月13日
個人情報保護責任者: ponpick
連絡先: ponpickkonpick@gmail.com

1. 収集される個人情報

本サービスは以下の情報を収集します。

  • 必須情報（会員登録/ログイン時）
    • メールアドレス（Google/Apple OAuthログイン）
    • ニックネーム（プロフィール名 or ユーザー入力）
  • サービス利用中に自動収集
    • 商品スキャン画像（ユーザー撮影）
    • レビュー内容（評価、テキスト、味タグ等）
    • Firebase Installation ID
    • アプリ使用ログ、エラー/クラッシュ情報
    • IPアドレス、OSバージョン、デバイス情報
  • 任意提供
    • 商品代表サムネイルとして使用する画像

正確な位置情報（GPS）は取得しません。

2. 利用目的

  • 会員管理、認証、問い合わせ対応
  • 商品スキャン、OCR認識、翻訳、検索・推薦
  • レビュー・写真を含むコミュニティ運営
  • サービス改善、障害解析、不正防止
  • 公式画像が不足する場合、ユーザー撮影画像を商品代表サムネイルとして利用

3. ユーザー生成コンテンツ（UGC）

  • ユーザーはアップロードした写真・レビューの著作権を保持します。
  • アップロードにより、PonPickに対し非独占的・無償・全世界的ライセンスが付与されます（表示・保存・サムネイル加工等）。
  • 一部スキャン画像が商品代表サムネイルに選定される場合があります。
  • 代表画像使用の停止はいつでも申請可能であり、当社は7日以内に削除します。

4. データ保管・削除

  • 目的達成後、または保有期間満了後、遅滞なく削除します。
  • アカウント削除時に即時削除される情報：
    • メールアドレス / ニックネーム / OAuth情報
    • 個人スキャン画像・履歴
    • Firebase Installation ID
  • レビュー及び評価はサービス信頼性維持のため匿名化して保管します。
  • 代表サムネイル画像は匿名のまま継続利用される場合があります（削除要求可能）。

5. 外部サービス・第三者提供

本サービスは次の事業者に処理を委託します：

  • Google Firebase（認証、DB、ログ/分析）
  • Google ML Kit / Cloud Vision（OCR処理）
  • Naver Papago API（翻訳）
  • Google / Apple OAuth（ログイン認証）

利用者の個人情報を第三者へ販売しません。

6. 国際データ移転（Cross-Border Transfer）

本サービスのデータの一部は国外サーバーで保存・処理されます。

移転先国：米国
移転される情報：メール、ニックネーム、スキャン画像、レビュー、ログ等
移転方法：HTTPS/TLS による暗号化通信
受領者：Google LLC（Firebase / Cloud Vision）
目的：認証、データ保存、OCR、障害解析
保管期間：アカウント削除まで、または処理完了まで
安全措置：暗号化保存、アクセス制御、GDPR整合措置

本サービスの利用により、国際データ移転に同意したものとみなします。

7. 利用者の権利

  • 個人情報の閲覧/訂正/削除/処理停止の要求
  • 同意撤回（アカウント削除含む）
  • 代表サムネイル画像の削除要求

8. セキュリティ対策

  • HTTPS/TLS暗号化通信
  • Firebase Security Rulesによるアクセス制御
  • Crashlyticsログ監視
  • バックアップ・侵入防止措置

9. 変更通知

重要な内容の変更がある場合、アプリ内告知またはメールで7日前までに通知します。
